The container that holds all your 3D objects
üì∑
CAMERA

// Create the scene
const scene = new THREE.Scene();

// Create a perspective camera
const camera = new THREE.PerspectiveCamera(
    75,                                    // Field of view
    window.innerWidth / window.innerHeight, // Aspect ratio
    0.1,                                   // Near clipping plane
    1000                                   // Far clipping plane

// Create the renderer

renderer.setSize(window.innerWidth, window.innerHeight);
document.getElementById('container').appendChild(renderer.domElement);

// Animation loop
function animate() {
    requestAnimationFrame(animate);
    renderer.render(scene, camera);
}

‚ú® Creating Particles

In Three.js, we use BufferGeometry and Points to create efficient particle systems. Each particle is just a point in 3D space.

particles.js JAVASCRIPT 
const particleCount = 3000;
const geometry = new THREE.BufferGeometry();

// Create arrays for particle attributes
const positions = new Float32Array(particleCount * 3);
const colors = new Float32Array(particleCount * 3);

// Fill with random positions
for (let i = 0; i < particleCount; i++) {
    positions[i * 3] = (Math.random() - 0.5) * 50;     // x
    positions[i * 3 + 1] = (Math.random() - 0.5) * 50; // y
    positions[i * 3 + 2] = (Math.random() - 0.5) * 50; // z
    
    // Random colors
    colors[i * 3] = Math.random();     // r
    colors[i * 3 + 1] = Math.random(); // g
    colors[i * 3 + 2] = Math.random(); // b
}

geometry.setAttribute('position', new THREE.BufferAttribute(positions, 3));
geometry.setAttribute('color', new THREE.BufferAttribute(colors, 3));

// Create material with additive blending for glow effect
const material = new THREE.PointsMaterial({
    size: 1,
    vertexColors: true,
    transparent: true,
    blending: THREE.AdditiveBlending
});

const particles = new THREE.Points(geometry, material);
scene.add(particles);
üí° Pro Tip

Using AdditiveBlending makes overlapping particles appear brighter, creating that beautiful glow effect!

üåÄ Shape Generation

The magic happens in how we position the particles. By using mathematical equations, we can create stunning shapes!

‚ù§Ô∏è Heart Shape

The heart uses a parametric equation. For each particle, we calculate its position based on a parameter t:

shapes.js JAVASCRIPT 
function generateHeartPositions(positions) {
    for (let i = 0; i < particleCount; i++) {
        const t = (i / particleCount) * Math.PI * 2;
        
        // Heart parametric equations
        const x = 16 * Math.pow(Math.sin(t), 3);
        const y = 13 * Math.cos(t) 
                  - 5 * Math.cos(2 * t) 
                  - 2 * Math.cos(3 * t) 
                  - Math.cos(4 * t);
        const z = (Math.random() - 0.5) * 8;
        
        positions[i * 3] = x;
        positions[i * 3 + 1] = y;
        positions[i * 3 + 2] = z;
    }
}

üåÄ Spiral Galaxy

For the galaxy, we create spiral arms by adding rotation based on distance from center:

shapes.js JAVASCRIPT 
function generateGalaxyPositions(positions) {
    const arms = 4;
    const radius = 30;
    
    for (let i = 0; i < particleCount; i++) {
        const r = Math.random() * radius;
        const armAngle = (i % arms) * (Math.PI * 2 / arms);
        const spin = r * 0.5;  // More spin as distance increases
        const angle = armAngle + spin;
        
        positions[i * 3] = Math.cos(angle) * r;
        positions[i * 3 + 1] = (Math.random() - 0.5) * 2;
        positions[i * 3 + 2] = Math.sin(angle) * r;
    }
}

‚úã Hand Tracking with MediaPipe

MediaPipe Hands detects 21 landmarks on each hand. We use these points to recognize gestures.

handTracking.js JAVASCRIPT 
const hands = new Hands({
    locateFile: (file) => {
        return `https://cdn.jsdelivr.net/npm/@mediapipe/hands/${file}`;
    }
});

hands.setOptions({
    maxNumHands: 2,
    modelComplexity: 1,
    minDetectionConfidence: 0.7,
    minTrackingConfidence: 0.5
});

hands.onResults((results) => {
    if (results.multiHandLandmarks) {
        for (const landmarks of results.multiHandLandmarks) {
            // Each landmark has x, y, z coordinates (0 to 1)
            const thumbTip = landmarks[4];
            const indexTip = landmarks[8];
            // ... detect gestures
        }
    }
});

Fingertips: Thumb=4, Index=8, Middle=12, Ring=16, Pinky=20

Wrist: 0

ü§ò Gesture Recognition

We detect gestures by checking the relative positions of finger landmarks:

gestures.js JAVASCRIPT 
function isThumbsUp(landmarks) {
    const thumbTip = landmarks[4];
    const thumbBase = landmarks[2];
    
    // Thumb is pointing up when tip is above base
    const thumbUp = thumbTip.y < thumbBase.y - 0.05;
    
    // Check other fingers are curled
    const indexCurled = landmarks[8].y > landmarks[6].y;
    const middleCurled = landmarks[12].y > landmarks[10].y;
    
    return thumbUp && indexCurled && middleCurled;
}

function isPeaceSign(landmarks) {
    // Index and middle extended, others curled
    const indexUp = landmarks[8].y < landmarks[6].y;
    const middleUp = landmarks[12].y < landmarks[10].y;
    const ringDown = landmarks[16].y > landmarks[14].y;
    const pinkyDown = landmarks[20].y > landmarks[18].y;
    
    return indexUp && middleUp && ringDown && pinkyDown;
}

üîó Putting It All Together

Finally, connect the gestures to particle behaviors:

main.js JAVASCRIPT 
function handleGesture(gesture) {
    switch (gesture) {
        case 'fist':
            particleSystem.setExpansion(0.3);  // Compress
            break;
        case 'palm':
            particleSystem.setExpansion(1.5);  // Expand
            break;
        case 'thumbsup':
            particleSystem.triggerBurst();   // Explosion!
            break;
        case 'peace':
            particleSystem.nextShape();       // Change shape
            break;
    }
}

üéì Pro Tips for Students

üí° Performance Optimization
  • Use BufferGeometry instead of regular Geometry
  • Limit particle count to 3000-5000 for smooth performance
  • Use needsUpdate = true only when positions change
üí° Gesture Detection Tips
  • Add a small threshold (0.02-0.05) to avoid flickering
  • Use gesture smoothing (track last 3-5 frames)
  • Test with different lighting conditions
‚ö†Ô∏è Common Mistakes to Avoid
  • Don't forget to request camera permissions
  • Always use HTTPS in production (required for camera access)
  • Handle the case when no hand is detected
